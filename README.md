# bunsen-yaru-gtk-fixed

This theme bundle was created to fix a few minor issues with the
bunsen-yaru-gtk project. This theme bundle includes unmodified versions
of all existing themes that came with the bunsen-yaru-gtk theme bundle
except for modified versions of the bunsen-yaru-dark and bunsen-yaru
themes that now have openbox-3 and xfce-notify-4.0 components, which
those 2 themes did not have previously (Meaning you could not use the
2 themes to the extent that you could use all the others).

The original yaru theme project which bunsen-yaru-gtk was based on:
https://github.com/ubuntu/yaru

bunsen-yaru-gtk: https://github.com/BunsenLabs/bunsen-yaru-gtk
